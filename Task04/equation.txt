1. Два разных корня
a=1, b=-3, c=2 → [1, 2]

2. Один корень
(дискриминант = 0)
a=1, b=2, c=1 → [-1]

3. Нет корней
(дискриминант < 0)
a=1, b=0, c=1 → []

4. Отрицательные коэффициенты
a=1, b=-5, c=6 → [2, 3]

5. Дробные корни
a=2, b=5, c=2 → [-2, -0.5]

6. Большие числа
a=1000, b=5000, c=6000 → [-3, -2]

7. b = 0
a=1, b=0, c=-4 → [-2, 2]

8. b и c равны нулю
(особый случай: x² = 0)
a = 1, b = 0, c = 0 → [0]

9. Очень маленькие коэффициенты
(проверка точности вычислений)
a = 0.001, b = 0.003, c = 0.002 → [-2, -1]
